# Andrew Ng 简介 (Andrew Ng)

- **身份**: Deeplearning.AI 与 Landing AI 创始人，Coursera 联合创始人，斯坦福大学兼职教授。
- **领域**: 机器学习教育、MLOps、工业落地。
- **贡献**: 早期百度首席科学家推动大规模语音/视觉深度学习；推出在线课程普及 ML；提出 “AI is the new electricity”。
- **立场**: 倡导务实落地、数据迭代驱动；对超级智能风险持谨慎但更关注现实应用。
- **作品**: 《Machine Learning Yearning》、Coursera ML / Deep Learning Specialization。
- **来源**: [Deeplearning.AI](https://www.deeplearning.ai/), [Coursera](https://www.coursera.org/instructor/andrewng), [Tech Review 采访 2015](https://www.technologyreview.com/2015/10/12/166103/what-its-like-to-teach-machine-learning-online/)
