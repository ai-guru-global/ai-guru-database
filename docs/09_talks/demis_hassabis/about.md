# Demis Hassabis 简介 (Demis Hassabis)

- **身份**: Google DeepMind 联合创始人兼 CEO。
- **贡献**: AlphaGo/AlphaFold 等里程碑式系统，推动强化学习+搜索+深度学习融合。
- **立场**: 认为 AGI 的目标是解决智能以解决其他问题；呼吁全球合作治理。
- **获奖**: 2018 年皇家学会院士；多项科学大奖。
