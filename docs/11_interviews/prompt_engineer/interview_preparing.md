# Prompt Engineer 面试准备

## 核心职责
- **提示词设计**: 任务分解、约束与质量控制。
- **应用落地**: 业务流程与模型能力对齐。
- **评测与优化**: 质量评测、错误分析与迭代。

## 核心能力
- **LLM 理解**: 模型能力边界与幻觉控制。
- **提示工程**: Few-shot、结构化输出、工具调用。
- **评测体系**: 任务指标与自动化测试。

## 常见考点
- **高质量 Prompt**: 可复用模板与策略。
- **失败案例分析**: 规则、示例与约束的改进。
- **业务适配**: 场景落地与效果评估。

## 项目准备
- **任务库**: 不同场景的 Prompt 模板与指标。
- **评测集**: 真实数据与对比实验。

## 简历要点
- **效果提升**: 命中率、准确率或转化提升。
- **可复用资产**: 模板与评测工具。
