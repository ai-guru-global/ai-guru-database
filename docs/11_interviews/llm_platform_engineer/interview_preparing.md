# LLM Platform Engineer 面试准备

## 核心职责
- **推理平台**: 模型服务化、路由与弹性扩缩。
- **性能优化**: 吞吐与延迟优化、缓存策略。
- **成本控制**: 资源调度与容量规划。

## 核心能力
- **系统设计**: 多模型版本管理、灰度发布。
- **推理加速**: 张量并行、KV Cache、量化。
- **平台工具**: 监控、计费与可观测性。

## 常见考点
- **高并发推理**: 请求调度与限流。
- **稳定性**: SLA、容错与回滚。
- **成本优化**: 资源利用率与成本指标。

## 项目准备
- **平台化案例**: 规模化模型部署与监控。
- **性能指标**: P99 延迟、吞吐、成本。

## 简历要点
- **工程规模**: 集群规模、服务 QPS。
- **优化成果**: 成本下降与性能提升。
