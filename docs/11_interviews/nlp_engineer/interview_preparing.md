# NLP Engineer 面试准备

## 核心职责
- **LLM 应用**: RAG、工具调用、对话系统。
- **文本建模**: 分类、生成、信息抽取。
- **评测与优化**: 评测集构建与性能提升。

## 核心能力
- **NLP 基础**: 分词、表示学习、Transformer。
- **工程能力**: 向量检索、索引与服务化。
- **评测能力**: 自动评测与人工评估结合。

## 常见考点
- **RAG 方案设计**: 索引、重排序、缓存。
- **LLM 对齐**: 提示词、微调与安全策略。
- **评测体系**: 精确率、召回率、幻觉处理。

## 项目准备
- **端到端 LLM 应用**: 需求 → 方案 → 指标。
- **性能与成本**: 延迟、吞吐与推理成本。

## 简历要点
- **效果提升**: 质量、覆盖率与业务增益。
- **可落地设计**: 架构与工程实现细节。
