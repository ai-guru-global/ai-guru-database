# 李飞飞简介 (Fei-Fei Li)

- **身份**: 斯坦福大学教授，斯坦福 HAI 联合主任，前 Google Cloud AI/ML 首席科学家。
- **领域**: 计算机视觉、认知神经科学、Human-Centered AI。
- **贡献**: ImageNet 发起人与负责人，推动深度学习在视觉领域的突破；倡导人本与负责任的 AI。
- **代表观点**: “AI is about people, not machines.” 强调以人为本、数据多样性、伦理和社会影响。
- **奖项/荣誉**: ACM/AAAI Fellow，《纽约时报》年度科技人物等。
- **来源**: [Stanford HAI](https://hai.stanford.edu/), [ImageNet 论文](https://www.image-net.org/), [TED 2017](https://www.ted.com/talks/fei_fei_li_how_we_teach_computers_to_understand_pictures)
