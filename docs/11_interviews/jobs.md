# AI 相关岗位与工种清单 (AI Roles & Jobs)

本清单面向“AI 方向求职/转岗/晋升”的学习与面试准备，覆盖研究、工程、数据、产品与安全方向。每个岗位均提供对应面试准备建议（见子目录）。

## 岗位地图

- **Machine Learning Engineer (机器学习工程师)**: 模型训练、特征工程、在线推理与服务化。
- **Data Scientist (数据科学家)**: 统计建模、实验设计、业务指标与洞察。
- **Research Scientist (研究科学家)**: 前沿算法研究、论文产出与学术影响。
- **Applied Scientist (应用科学家)**: 研究成果落地、模型性能到业务指标的桥梁。
- **MLOps Engineer (机器学习运维工程师)**: 模型训练/发布流水线、监控与稳定性。
- **Data Engineer (数据工程师)**: 数据管道、数据质量、离线/实时特征建设。
- **NLP Engineer (自然语言处理工程师)**: LLM 应用、RAG、文本模型与评测。
- **Computer Vision Engineer (计算机视觉工程师)**: 视觉模型训练、检测/分割/生成。
- **AI Product Manager (AI 产品经理)**: 需求定义、产品策略、模型能力到用户价值。
- **AI Security Engineer (AI 安全工程师)**: 安全评测、红队、提示词注入防护。
- **AI Research Engineer (AI 研究工程师)**: 研究工程化、实验复现与性能优化。
- **AI Research Scientist (AI 研究科学家)**: 研究创新、论文产出与学术影响。
- **LLM Platform Engineer (大模型平台工程师)**: 推理平台、路由与成本优化。
- **Prompt Engineer (提示词工程师)**: 提示策略、评测与业务落地。
- **AI Infrastructure Engineer (AI 基础设施工程师)**: 训练基础设施与集群稳定性。
- **AI Reliability Engineer (AI 可靠性工程师)**: 线上稳定性、监控与恢复。
- **AI Solutions Architect (AI 解决方案架构师)**: 端到端方案设计与交付。
- **AI Data Analyst (AI 数据分析师)**: 指标体系、A/B 测试与业务洞察。
- **AI Policy Specialist (AI 政策与合规专家)**: 法规合规、治理与风险评估。
- **AI Evaluation Engineer (AI 评测工程师)**: 评测基准、质量门禁与回归测试。
- **Robotics Engineer (AI 机器人工程师)**: 感知、规划与控制系统集成。

## 使用建议

1. 先选择目标岗位，阅读其目录中的 `interview_preparing.md`。
2. 按“核心能力 → 项目经验 → 系统设计 → 题库与案例”顺序准备。
3. 建议结合目标公司 JD 与团队技术栈进行定制化补充。
