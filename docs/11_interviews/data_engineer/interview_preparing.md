# Data Engineer 面试准备

## 核心职责
- **数据管道**: 离线/实时数据 ETL 与数据湖/仓。
- **数据质量**: 校验、治理与元数据管理。
- **特征工程**: 特征存储与服务化。

## 核心能力
- **数据系统**: Spark/Flink、Kafka、Airflow。
- **数据建模**: 维度建模、指标口径设计。
- **性能优化**: 分区、索引与并行计算。

## 常见考点
- **数据一致性**: 实时与离线对齐。
- **数据质量**: 缺失、异常与漂移处理。
- **成本优化**: 计算与存储成本控制。

## 项目准备
- **大规模数据案例**: 流量、延迟与稳定性数据。
- **数据治理实践**: 元数据、血缘与权限。

## 简历要点
- **数据规模与性能**: QPS、延迟、吞吐量。
- **业务贡献**: 数据平台支撑的业务价值。
