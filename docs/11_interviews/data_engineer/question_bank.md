# Data Engineer 题库

## 基础
- 解释 ETL 与 ELT 的区别与适用场景。
- 数据建模中维度建模的核心思想是什么？
- 如何保证数据质量与一致性？

## 项目
- 描述一个实时数据管道的设计与落地。
- 如何构建特征存储并服务化？
- 如何处理离线与实时数据的一致性？

## 系统设计
- 设计一个批流一体的数据平台架构。
- 数据治理、血缘与权限体系如何设计？
- 如何进行数据成本优化？

## 案例
- 数据延迟突然升高如何排查？
- 上游数据格式变化导致故障如何处理？
- 数据缺失或重复如何修复？
