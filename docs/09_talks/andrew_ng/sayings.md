# Andrew Ng 关于 AI 的观点与格言

> 说明：以下整理 Andrew Ng 在公开场合关于 AI 的代表性言论，提供中英文对照与出处链接。

## 1. 实用主义与落地 (Pragmatism & Deployment)

1. **“AI is the new electricity.” / “AI 是新的电力。”**  
   - **上下文**: 强调 AI 将像电力一样赋能各行各业。  
   - **来源**: [Stanford AI for All Keynote, 2017](https://hai.stanford.edu/events)

2. **“Worrying about superintelligence is like worrying about overpopulation on Mars.” / “担心超级智能就像担心火星人口过剩。”**  
   - **上下文**: 呼吁优先解决现实可落地的问题。  
   - **来源**: [Interview with Andrew Ng, 2015](https://www.technologyreview.com/2015/10/12/166103/what-its-like-to-teach-machine-learning-online/)

## 2. 数据与迭代 (Data & Iteration)

3. **“Data is the new oil.” / “数据是新的石油。”**  
   - **上下文**: 强调数据质量与规模对模型性能的关键性。  
   - **来源**: [Coursera Machine Learning Courses, 常被引用的比喻](https://www.coursera.org/learn/machine-learning)

4. **“Move fast, iterate, but also measure.” / “快速行动、迭代，但一定要量化测量。”**  
   - **上下文**: 提倡用实验与指标驱动的工程文化。  
   - **来源**: [DeepLearning.AI / Landing AI 公开访谈](https://www.deeplearning.ai/blog/)

## 3. 教育与人才 (Education & Talent)

5. **“Everyone can learn AI.” / “人人都能学 AI。”**  
   - **上下文**: 鼓励大众通过在线教育掌握 AI 能力。  
   - **来源**: [Coursera Machine Learning & DeepLearning.AI](https://www.coursera.org/specializations/deep-learning)
