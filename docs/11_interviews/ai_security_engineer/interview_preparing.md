# AI Security Engineer 面试准备

## 核心职责
- **安全评测**: 红队测试、越狱与提示词注入检测。
- **风险控制**: 过滤、审计与安全策略制定。
- **防护建设**: 模型安全、数据安全与合规。

## 核心能力
- **攻击与防御**: Prompt Injection、越狱、对抗样本。
- **评测体系**: 安全基准与自动化测试。
- **工程落地**: 策略引擎、风险监控与响应。

## 常见考点
- **安全评测方法**: 指标设计与回归测试。
- **防护方案**: 规则、模型与多层防御。
- **合规理解**: 数据合规、内容安全。

## 项目准备
- **安全平台或流程**: 评测 → 修复 → 复测闭环。
- **案例复盘**: 真实攻击与防御效果。

## 简历要点
- **风险降低**: 越狱率下降、违规输出降低。
- **防护体系**: 多层防御策略与治理流程。
