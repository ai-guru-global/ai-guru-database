# Computer Vision Engineer 面试准备

## 核心职责
- **视觉模型训练**: 分类、检测、分割与生成。
- **数据与标注**: 数据治理与标注质量控制。
- **部署与优化**: 端侧与云侧推理优化。

## 核心能力
- **视觉模型**: CNN、ResNet、ViT、YOLO。
- **训练技巧**: 数据增强、迁移学习、蒸馏。
- **推理优化**: 量化、裁剪与加速。

## 常见考点
- **模型选择**: 任务需求与架构权衡。
- **数据问题**: 类别不平衡、噪声与漂移。
- **部署策略**: 端云协同与延迟优化。

## 项目准备
- **真实场景项目**: 工业检测、安防、医疗影像等。
- **效果指标**: mAP、IoU、F1、延迟。

## 简历要点
- **结果量化**: 准确率与成本/速度提升。
- **全链路经验**: 数据 → 训练 → 部署。
