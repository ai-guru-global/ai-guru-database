# AI Reliability Engineer 面试准备

## 核心职责
- **系统可靠性**: 推理稳定性、故障恢复。
- **质量保障**: 模型性能回归与监控体系。
- **风险控制**: 线上异常与安全策略。

## 核心能力
- **SRE 思维**: SLI/SLO、故障演练与复盘。
- **模型监控**: 漂移检测、异常报警。
- **自动化**: 回滚、降级与容灾。

## 常见考点
- **监控指标**: 延迟、准确率与漂移。
- **故障处理**: 线上异常定位与修复。
- **稳定性方案**: 灰度、容灾与回滚策略。

## 项目准备
- **稳定性案例**: 线上故障复盘与修复。
- **质量体系**: 回归测试与评测基准。

## 简历要点
- **风险降低**: 故障率下降与恢复时间缩短。
- **自动化体系**: 工具与平台建设。
