# AI Infrastructure Engineer 面试准备

## 核心职责
- **训练基础设施**: 计算资源调度与集群管理。
- **数据与存储**: 高性能数据管道与存储架构。
- **可靠性**: 训练与推理平台稳定性。

## 核心能力
- **分布式系统**: 通信、容错与一致性。
- **性能优化**: I/O、网络与存储优化。
- **平台建设**: 统一的训练与服务平台。

## 常见考点
- **调度与隔离**: 资源配额与多租户。
- **性能瓶颈**: I/O 与网络排查。
- **高可用设计**: 故障恢复与自动化运维。

## 项目准备
- **平台建设**: 资源调度或训练平台案例。
- **优化结果**: 吞吐、成本与稳定性改进。

## 简历要点
- **平台规模**: GPU 数量、集群规模。
- **性能与稳定性**: SLA 与成本指标。
