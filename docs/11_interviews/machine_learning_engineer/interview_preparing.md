# Machine Learning Engineer 面试准备

## 核心职责
- **模型训练与评估**: 数据清洗、特征工程、训练与离线评估。
- **在线推理与服务化**: 推理性能、可用性与稳定性优化。
- **端到端落地**: 从实验到部署的工程化能力。

## 核心能力
- **算法基础**: 监督/无监督学习、损失函数与优化。
- **工程能力**: Python/C++、数据管道、模型服务。
- **系统设计**: 低延迟推理、A/B 实验、监控与回滚。

## 常见考点
- **模型与指标**: AUC/PR、偏差-方差权衡、过拟合处理。
- **训练技巧**: 正则化、早停、特征选择。
- **线上问题**: 数据漂移、冷启动、分布偏移。

## 项目准备
- **准备 2-3 个端到端项目**: 数据 → 训练 → 部署 → 监控。
- **量化成果**: 延迟、成本、效果提升的真实数字。

## 简历要点
- **业务指标提升**: 转化率、召回率、成本下降等。
- **可复现与可落地**: 明确数据规模、模型规模与工程链路。
