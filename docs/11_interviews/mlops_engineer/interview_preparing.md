# MLOps Engineer 面试准备

## 核心职责
- **训练与部署流水线**: CI/CD、模型版本管理。
- **监控与稳定性**: 数据漂移、模型退化与回滚。
- **系统效率**: 资源调度、成本与性能优化。

## 核心能力
- **工程系统**: Docker/Kubernetes、CI/CD。
- **模型生命周期**: 训练、评估、发布、监控。
- **可观测性**: 日志、指标、告警与追踪。

## 常见考点
- **模型漂移检测**: 指标变化与触发策略。
- **上线策略**: 灰度、回滚与多版本管理。
- **成本优化**: 资源利用率与推理成本控制。

## 项目准备
- **端到端 MLOps 案例**: 流水线设计与故障处理。
- **系统设计题**: 高可用、低延迟的推理服务。

## 简历要点
- **稳定性与成本**: SLA、P99 延迟、资源节省。
- **可复用平台**: 你搭建的工具与平台能力。
