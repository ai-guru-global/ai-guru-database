# Geoffrey Hinton 简介 (Geoffrey Hinton)

- **身份**: 深度学习先驱，图灵奖得主（2018），前 Google Brain 研究员，多伦多大学荣休教授。
- **贡献**: 反向传播、深度信念网络、Capsule Network 等奠基性工作；指导 ImageNet 2012 胜利（AlexNet）。
- **立场**: 近年强调 AGI 潜在风险，呼吁安全与治理；“如果我不做，总会有人做”。
- **影响**: 被称为“深度学习教父”，对现代神经网络架构与训练方法影响深远。
- **来源**: [NYT 采访 2023](https://www.nytimes.com/2023/05/01/technology/geoffrey-hinton-google-artificial-intelligence.html), [Turing Award 2018](https://amturing.acm.org/award_winners/hinton_3956500.cfm)
