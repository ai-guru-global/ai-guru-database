# Data Scientist 面试准备

## 核心职责
- **统计建模与分析**: 建模、实验设计与指标解释。
- **业务洞察**: 发现机会、输出可执行策略。
- **实验与因果**: A/B 测试、因果推断与效果评估。

## 核心能力
- **统计基础**: 置信区间、假设检验、回归分析。
- **数据处理**: SQL、特征构造、异常处理。
- **业务理解**: 指标体系、用户行为与增长机制。

## 常见考点
- **实验设计**: 样本量、功效分析、干预偏差。
- **统计推断**: p-value、因果关系与混杂因素。
- **模型解释**: 可解释性与业务落地。

## 项目准备
- **准备 2-3 个分析项目**: 问题定义 → 方法 → 结论 → 影响。
- **聚焦业务价值**: 决策支持、增长/留存提升。

## 简历要点
- **清晰的业务目标**: 用数据驱动决策。
- **结果导向**: 指标变化与可量化收益。
