# AI Evaluation Engineer 面试准备

## 核心职责
- **评测体系**: 评测基准、指标与自动化测试。
- **质量保障**: 回归评测与上线前验收。
- **错误分析**: 失败案例与改进建议。

## 核心能力
- **评测方法**: 自动评测 + 人工评估结合。
- **指标设计**: 质量、覆盖率、鲁棒性。
- **工程能力**: 数据标注、评测平台构建。

## 常见考点
- **评测设计**: 数据集构建与偏差控制。
- **一致性**: 评分标准与评审一致性。
- **线上对齐**: 离线指标与线上表现映射。

## 项目准备
- **评测平台**: 数据→评测→报告闭环。
- **案例复盘**: 评测推动模型优化。

## 简历要点
- **评测影响力**: 指标提升与质量改进。
- **体系化能力**: 评测标准与流程建设。
